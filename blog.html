Important FAQ

What is the Document Object Modal (DOM)?

The Document Object Model (DOM) is like a map of a web page that lets you change stuff—like text, images, and buttons—using code. It helps browsers understand and interact with the page, making it easy to update how things look and work.

How do you select a Element from DOM?

To select an element from the DOM, you can do a few things: Use getElementById("id") to get an element by its ID. Try querySelector("selector") for more flexibility with CSS selectors. Use getElementsByClassName("class") to grab all elements with a specific class. Or go with getElementsByTagName("tag") to pick elements by their tag. It's like choosing your favorite items from a shop!

What is event delegation in the context of DOM, and why is it useful?

Event delegation is when you put one event listener on a parent element to handle events for all its kids. It’s super handy because it saves memory and still works for any new elements you add later. Keeps things neat and efficient!

How do you manipulate an element's attributes and styles?

To change an element's attributes and styles, just grab it by its ID and update what you need. It’s pretty straightforward!
